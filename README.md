# Claude Skills Marketplace

Personal Claude Code skill marketplace — 12 skill packs, 200+ individual skills.

## Install as a marketplace

```bash
claude plugin marketplace add alim-c3/claude-skills
```

## Install individual packs

```bash
claude plugin install business-growth@alim-c3-skills
claude plugin install c-level-advisor@alim-c3-skills
claude plugin install engineering@alim-c3-skills
# etc.
```
